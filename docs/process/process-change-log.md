# Atlas 流程变更台账

本文件记录 AI 协作开发流程的正式变更。任何对流程、模板、提示词、检查清单的调整，都应在这里留下原因和生效范围。

## 记录规则

1. 只记录已经决定生效的流程变更，不记录讨论草稿。
2. 每条变更必须说明原因、影响范围、owner 和验证方式。
3. 涉及安全、权限、数据库、生产配置、发布门禁的变更，必须由 AI 代码守护人确认。
4. 变更后需要同步更新对应文档，例如：
   - `docs/process/atlas-ai-development-workflow.md`
   - `docs/requirements/REQUIREMENT_TEMPLATE.md`
   - `docs/prompts/`
   - `docs/qa/`
